# Rock Paper Scissors

A fun **Rock-Paper-Scissors game** powered by **OpenCV** and **MediaPipe Hands**.  
This project uses **real-time hand gesture recognition** to detect whether the player shows **Rock**, **Paper**, or **Scissors**, and plays against the computer with a simple on-screen UI.

## Tech Stack
- **Python 3**
- **OpenCV**
- **MediaPipe**

---

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/rock-paper-scissors-cv.git
   cd rock-paper-scissors-cv
   ```
1. Install dependencies:
   ```bash
   pip install opencv-python mediapipe
   ```
1. Run the game:
   ```bash
   python rock_paper_scissors.py
   ```

## How to Play
Press P to start the countdown.
Show one of the following gestures to the camera:
✊ Rock – All fingers folded
✋ Paper – All fingers open
✌️ Scissors – Only index and middle fingers open
Wait for the result to appear!
Press P again to play another round or Q to quit.
